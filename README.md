## 🔍 Overview

**WaltonPlaza Automation Framework** is a professional-grade test automation solution designed to validate the core functionalities of the Walton Plaza web application. Built on the **Page Object Model (POM)** design pattern, the framework provides a clean separation between test logic and UI interactions — making tests highly readable, reusable, and easy to maintain.

The framework covers critical user journeys including **Login**, **Home Page**, and **Dashboard** flows, with full test lifecycle management via TestNG and rich visual reporting powered by Allure Reports.

---

## ✨ Features

- ✅ **End-to-End Web Automation** — Covers complete user journeys across the Walton Plaza application
- 🏗️ **Page Object Model (POM)** — Clean architecture separating page interactions from test logic for maximum reusability
- 🔐 **Automated Login Flow** — Validates authentication scenarios including positive and edge cases
- 🏠 **Home Page Testing** — Automated verification of home page components and navigation
- 📊 **Dashboard Testing** — Validates dashboard data rendering and UI interactions
- 🧪 **TestNG Suite Execution** — Organized test suites with grouping, sequencing, and dependency management
- 📸 **Automatic Screenshot Capture** — Screenshots captured on both test pass and failure for evidence
- 📈 **Allure Reporting** — Detailed, interactive HTML reports with logs, screenshots, and step-by-step breakdowns
- ⚙️ **Maven Build Integration** — Simple, portable build and execution via Maven commands
